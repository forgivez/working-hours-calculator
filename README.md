# ⏰ Working Hours Calculator

출근 시간 기준으로 **퇴근 시간까지 남은 시간을 초 단위로 실시간 표시**해주는 데스크탑 앱입니다.  
점심시간 1시간을 포함하여 **총 근무시간 9시간 기준**으로 퇴근 시간을 계산합니다.

## 🖥️ 주요 기능

- 출근 시간 입력 (예: `09:00`)
- 총 9시간 기준 퇴근 시간 자동 계산
- **실시간 카운트다운** (`시:분:초`)
- 간결하고 감성적인 GUI (`tkinter`)
- `.exe` 실행파일로 배포 가능 (Python 미설치 환경에서도 실행됨)
